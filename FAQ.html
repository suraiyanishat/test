Ans:An Internet Protocol (IP) address is a unique numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication. An IP address serves two main functions: Identification: Identifies the host or network interface. Location Addressing: Provides the location of the host in the network. IP addresses are of two types: IPv4 (e.g., 192.168.1.1) and IPv6 (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

Ans:Computer Hardware: The physical components of a computer system. Examples include the CPU, RAM, hard drive, motherboard, monitor, keyboard, and mouse. Hardware is tangible and can be physically touched and seen. Computer Software: The set of instructions, programs, or data that operate the hardware. Examples include operating systems (e.g., Windows, macOS, Linux), applications (e.g., Microsoft Office, web browsers), and system software. Software is intangible and cannot be physically touched; it exists as code.
